Detecting Sound

  1. Now we're going to make a function to make the sound work. Simply drag the on loud sound command block and put the the variable block set on with the parameter not on inside of the variable block, which should be inside of the on loud sound block. Then add in an if else block underneath the variable block inside of the on loud sound block.

  2. Then click on the plus sign of the if else block to add an else if statement to the block.

  3. After you have added an else if statement to the block remove the else line so that you can only see the if and else if statements in the block.

  4. Then add in the variables on and not on to the if and else if statement. Further more, add in the sound blocks, which will play the specific sound when the circuit turns on or off.
