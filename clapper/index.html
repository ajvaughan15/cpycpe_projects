What You Will Do

You will learn to make a clapper on the Circuit Playground Express. Meaning whenever you clap your hands together or make a loud noise your CPE will turn on or off depending what state it is in.

What You Will Learn

  • Use block commands to code CPE in adafruit
  • How to make parameters
  • How to make a function that uses the proper parameters
  • How to return a function and implement an animation

To start things off, start by clicking here. It will take you to the adafruit website where you can make new projects, by pressing New Project.
